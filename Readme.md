# Use with M1/M2 Based Kali

Select JDX 11:
	1. git clone https://github.com/jcf-junior/Burp-Suite-m1-mac.git
	2. sudo update-alternatives --config java
	3. cd Burp-Suite/
	4. sudo ./Kali_Linux_Setup.sh
	
## To start Burp in case it doesn't auto start:
	1. burp
